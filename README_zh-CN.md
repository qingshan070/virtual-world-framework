
# VWI：虚拟世界框架

[English](README.md) | [简体中文](README_zh-CN.md)

> 没有自由意志，只有应力、形变与涌现所构成的因果链。

VWI 是一个**研究导向的虚拟世界框架**，用于探索如何将决定论模拟、人格建模与大语言模型结合为一个统一的沙盒系统。

本仓库目前处于 **early alpha / 概念验证** 阶段，当前公开的内容主要包括：

- 核心理论文档与白皮书
- 一个早期的 Python 双引擎 demo
- 若干面向后续迭代的实现说明

> **命名说明**
>
> 本仓库当前托管名称为 `virtual-world-framework`。  
> `VWI` 仍是现阶段在既有文档与 demo 代码中继续使用的项目内部代号。

---

## VWI 想解决什么问题

VWI 试图把角色理解为**承受应力的材质**，而不是一组静态的人格标签。

在这套框架里：

- 外部事件被视为多维度冲击
- 角色通过内部执念结构对冲击进行放大或过滤
- 心理变化以弹性形变或塑性形变的方式累积
- 决策来自确定性的状态转移，而不是一个抽象的“自由意志”
- 角色行为会写回世界，并继续生成新的因果链条

这个项目想做的，不是用模糊的“性格设定”去假装人物真实，而是搭建一个在**内在认知**与**外部世界状态**之间可计算、可闭环的模拟结构。

---

## 核心结构

VWI 目前围绕一个“双引擎”结构展开：

### 1）有情世间（Sentient World）
角色内部的认知与心理层。

这一层当前主要关注：

- 主观应力计算
- 记忆与创伤累积
- 弹性 / 塑性心理形变
- 极端压力下的相变
- 有边界的决策输出

### 2）器世间（Vessel World）
角色之外的外部世界层。

这一层当前主要关注：

- 事件生成与扩散
- 环境变化
- 业力 / 后果写回
- world state 在 tick 间的演化

两层共同形成一个闭环：

**世界冲击角色 -> 角色发生变化 -> 角色采取行动 -> 世界再次改变**

---

## 理论方向

VWI 采用的是一种**人格材料力学**视角。

其中的关键概念包括：

- **应力**：来自肉体、物质、社会、精神等维度的外部冲击
- **弹性形变**：暂时性的情绪波动
- **塑性形变**：不可逆的人格扭曲或创伤累积
- **断裂**：长期超载后的彻底崩溃
- **相变 / 合金化**：在极端条件下，角色底层材质结构可能发生改变

这是一种研究模型，不是对真实心理学的直接宣称。  
当前实现的重点，是验证这样一套框架能否产生自洽的模拟行为。

## 当前状态

这个仓库**不是**一个成熟可商用的引擎产品。

你现在看到的内容，更适合被理解为：

- 一个研究 alpha
- 一套概念框架
- 一个早期技术原型
- 一个持续演化中的公开起点

有些部分目前是有意简化的。  
例如，当前 demo 中的人口初始化与社会拓扑仍然偏占位性质，主要目的是证明整条引擎管线能够从输入跑到输出，而不是声称已经实现了高度真实的复杂社会动力学。
