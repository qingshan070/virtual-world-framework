# VWI (Virtual World Infrastructure) : 有情世间与器世间双轨引擎

[English Version](README.md) | [中文版](README_zh-CN.md)

> **"没有自由意志，只有应力、形变与涌现的因果。"**

VWI (Virtual World Infrastructure) 是一个**研究型虚拟世界基础设施**，旨在通过“物理引擎+大语言模型”的双向绑定，构建一个严格遵循决定论、具有微观心理演化能力和宏观社会涌现属性的自治沙盒系统。

本项目目前处于 **Research Alpha** 阶段，主要开源了其底层的**核心理论架构与白皮书**，并附带了部分概念验证代码 (`vwi_demo`)。

---

## 核心本体论：无我，但有轮回

本系统的哲学内核深受佛教本体论启发，并将其转化为严谨的软件工程范式：

在这个世界中，**NPC (Character) 没有所谓的“自由意志”**。
角色的每一个决策、每一句台词，都是其“人格材质”在面对外部世界“物理冲击”时，产生的**确定性形变计算结果**。
- **无我**：没有一个恒常的、拥有自由意志的“灵魂”在做决定。角色只是一组不断被外界事件捶打、扭曲、退火的“材料属性”的集合（刚度 $E$、韧性 $T$、塑性形变 $\varepsilon_p$）。
- **轮回 (Karma)**：角色的行为产生“业力 (Karma)”，写回并改变外部世界（器世间），从而引发新的事件，继续冲击其他角色的心理材质，形成不灭的因果链。

系统通过极其复杂的计算闭环，让这套确定性的方程，涌现出了看起来像是有“选择”、有“人性”的**自由意志幻觉**。

## 理论基石：材料物理学映射心理学

系统摒弃了传统的“性格标签”和“线性扣血”模型，首创了**人格材料力学模型 (Personality Material Mechanics)**：

1. **应力 (Stress) $\sigma$**：外界事件造成的四大维度（肉体、物质、社会、精神）冲击，经过角色的“执念矩阵”放大，形成心理压强。
2. **弹性形变 (Elastic Strain) $\varepsilon_e$**：暂时的情绪波动，压力消失后会平复。
3. **塑性形变 (Plastic Strain) $\varepsilon_p$**：不可逆的性格扭曲。当压力超过角色的“屈服点”时，角色产生永久的心理创伤和性格偏移。
4. **断裂 (Fracture)**：当塑性形变越过极值，角色将彻底崩溃。

### 灵魂炼金术：相变与合金化
在极端的心理形变下，大语言模型 (LLM) 将作为“炼金术士”介入，为角色注入合金元素，改变其底层材质结构（相变）：
- 注入 **碳 (C)**：变硬、变脆，走向傲慢或黑化（马氏体）。
- 注入 **镍 (Ni)**：变得心软，提升韧性，走向包容或圣徒化（奥氏体）。
- 注入 **硫 (S)**：产生毒性，急剧降低崩溃阈值。

## 工程闭环：七步演算状态机 (FSM)

角色的每一次认知循环，都必须严格穿过以下基于“五蕴（色受想行识）”设计的七步管线：

1. **触 (Contact)**：感知与多维主观应力计算（客观冲击 $\times$ 执念放大）。
2. **缘 (Conditioning)**：情境缓冲与历史创伤锚点的应力集中效应 ($K_t$)。
3. **受 (Sensation)**：计算弹性形变与塑性扭曲累积。
4. **想 (Cognition)**：动机生成，并在极端压力下触发 LLM 的合金化判定。
5. **行 (Conflict)**：计算内在驱动力与道德/风险阻力的摩擦力 ($F_{int}$)。
6. **识 (Decision)**：五态决策输出（系统强制收敛为 `EXPLODE`, `ACT`, `HESITATE`, `SUPPRESS`, `BREAKDOWN` 五大绝对枚举）。
7. **业 (Karma)**：多维业力写回，影响外部世界并记录时间衰减日志。

## 架构保证：Tick隔离与单一真相源

为了确保系统在多角色并发下的因果确定性，VWI 采用了严格的工业级架构规范：
- **Tick 隔离 (Double Buffering)**：在任意 Tick $N$ 中，角色只能读取被冻结的只读快照 $State_N$。
- **CQRS 与事务池**：角色在“业”阶段产生的决策，被统一推入全局事务池，由 `Karma Resolver` 统一处理冲突合并后，生成 $State_{N+1}$。
- **LLM 结构化钳制**：通过严格的 JSON Schema 与全局枚举真相表，彻底封死 LLM 产生数值发散或语义漂移的风险。

## 开源内容

本仓库目前包含以下核心内容：

1. **理论白皮书** (`*.md`): 包含完整的基础设施架构设计、有情世间（心理引擎）与器世间（物理沙盒）的详细数学公式与工程流转图。这是本项目的绝对地基。
