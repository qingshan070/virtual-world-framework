
# VWI：虚拟世界基础设施 (概念验证原型)

[English Version](README.md) | [中文版](README_zh-CN.md)



VWI 是一个面向研究的虚拟世界基础设施。它探索如何将决定论模拟、人格建模与大语言模型（LLM）结合成一个统一的沙盒系统。它不把角色视为静态的人格标签，而是将其视为受压状态下的材料。外部事件是多维度的冲击，角色通过内在的执念结构对这些冲击进行转化。心理变化会以弹性形变或塑性形变的方式累积，决策则源自决定论的状态转移，而非传统意义上的自由意志。角色的行为会写回世界，并产生新的因果链。

## 双引擎结构

该框架运行在双引擎循环之上：

1. **有情世间 (Sentient World)**
   - 角色内部认知层
   - 主观应力计算
   - 记忆与创伤累积
   - 弹性与塑性心理形变
   - 极端压力下的相变
   - 有界决策输出

2. **器世间 (Vessel World)**
   - 外部世界层
   - 事件生成与传播
   - 环境演变
   - 业力/后果写回
   - 跨 Tick 的世界状态演化

## 理论方向：人格材料力学

VWI 采用了“人格材料力学” (personality material mechanics) 的视角。关键概念包括：

- **应力 (Stress)**：由外部事件对角色施加的多维度心理压强，会通过角色特定的执念被放大。
- **弹性形变 (Elastic strain)**：暂时的心理或情绪波动，会随时间恢复。
- **塑性形变 (Plastic strain)**：永久的、不可逆的心理扭曲或创伤。
- **断裂 (Fracture)**：当塑性形变超过临界阈值时发生的彻底心理崩溃。
- **相变 / 合金化 (Phase transformation / alloying)**：在极端压力下触发的人格底层结构的根本性转变（例如变得冷酷或变得包容），有时会有 LLM 介入辅助。

请注意：这是一种研究模型，并不宣称实现了绝对的心理学拟真。当前的实现主要用于测试这样一套基础设施能否产生连贯的模拟行为。

## 当前状态

这个仓库**不是**生产级引擎。它是一个研究阶段的 alpha 版本、一个概念框架、一个早期的技术原型，以及一个不断演进系统的公开快照。

许多组件是故意简化的。例如，当前的演示代码使用了占位符风格的随机人口生成和简化的社会拓扑，这主要是为了证明引擎管线可以端到端跑通。

## 运行演示

当前的 Python 演示使用了包风格的相对导入。因此，应该在项目根目录下以模块方式运行，而不是直接执行 `__main__.py`。

请使用以下命令运行模拟：

```bash
python -m vwi_demo --ticks 20 --mode GOD
```

### 可用参数：
- `--ticks`: 运行的 Tick 数量。
- `--mode`: 交互模式，可选 `GOD`、`ARCHON` 或 `AVATAR`。
- `--seed`: 用于随机生成的整数种子。
- `--use-llm`: 启用 LLM 集成的标志。LLM 辅助行为需要配置 API Key；否则，演示将优雅地退化为纯数学/非 LLM 路径。
- `--interactive`: 启用逐步交互模式的标志。
- `--model`: 指定使用的 LLM 模型名称 (默认: `gpt-3.5-turbo`)。
- `--avatar`: 在 `AVATAR` 模式下扮演的角色 ID。
- `--population`: 初始随机生成的节点数量 (默认: 4)。

运行日志会写入：`outputs/run_log.jsonl`

## 演示说明

当前的演示应被理解为一个引擎实验，而不是一个完整的社会模拟产品。这一版主要用于验证：
- 双引擎循环
- 应力 -> 形变 -> 决策的映射关系
- 世界写回逻辑
- 基础的 LLM 辅助相变流程

它**还不代表**：
- 一个完全拟真的社会模型
- 一个大规模并发架构
- 一个生产部署系统
