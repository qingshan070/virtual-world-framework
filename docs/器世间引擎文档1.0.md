## 摘要

器世间引擎是虚拟世界的宏观物理层，负责模拟"世界容器"的运行。其核心哲学基于佛教宇宙观的"四大"（地水火风）与"成住坏空"法则，将复杂的世界生态简化为四维参数的熵增过程。本引擎不包含任何目的论或心理学，仅执行冷酷的物理法则：生成资源、维持稳态、推动腐烂、触发相变。

## 一、核心理念

### 1.1 器世间的定义

在佛教宇宙观中，"器世间"（Vessel World）指承载有情众生的物质世界。"器"即容器，世界只是一个被动的容器，没有意识，没有目的，不会偏袒任何人。

**工程意义**：世界引擎不需要模拟复杂的心理或社会目的，只需要执行物理法则。

### 1.2 四大皆空，但四大皆是参数

世界的底层实体不是树木、城墙或金钱，而是四个维度的基础变量：

| 四大 | 梵文 | 物理语义 | 系统参数 |
|------|------|---------|---------|
| **地** | Pṛthivī | 坚固性/存量 | 一切静态的、有质量的、占据空间的资源 |
| **水** | Āpas | 湿润与凝聚/连接 | 一切负责连接和传输的管道 |
| **火** | Tejas | 温度与转化/动能 | 系统的消耗率和转化率 |
| **风** | Vāyu | 流动与传播/扩散 | 无质量物体的扩散引擎 |

### 1.3 成住坏空：熵增法则

世界运行的唯一方向是"熵增"，在东方哲学中表现为"成住坏空"：

| 阶段 | 语义 | 物理意义 | 系统行为 |
|------|------|---------|---------|
| **成** | 生成 | 负熵注入 | 资源生成、秩序建立 |
| **住** | 维持 | 稳态期 | 短暂的平衡 |
| **坏** | 衰败 | 熵增法则 | 资源腐烂、秩序崩解 |
| **空** | 重置 | 相变 | 系统崩溃与重构 |

## 二、数据模型

### 2.1 世界状态向量

```

  Earth:   { food, water, energy, gold, iron, land, weapon, ... },

  Fire:    { consumption_rate, decay_rate, conflict_intensity, metabolism },
  Wind:    { memes, info_temperature, topic_salience, rumor_spread }

```

### 2.2 地（Earth）- 资源维度的张量化 (ResourceDim)

**定义**：世界的基础存量，不再使用具象的“粮食”、“金钱”等概念，而是抽象为衡量生存空间的四大基本维度。

**四大维度 (ResourceDim)**：

| 维度 | 英文 | 物理/社会语义 | 典型具象体现 |
|------|------|-------------|------------|
| **肉体/生存** | SOMATIC | 生命的基本存续能力 | 粮食、水、健康、寿命、免于杀戮的物理安全 |
| **物质/经济** | MATERIAL | 生产资料与交易媒介 | 黄金、土地、房屋、生产工具 |
| **社会/权力** | SOCIAL | 社会结构中的位置与影响力 | 官职、科举名额、宗族地位、人脉、阶级特权 |
| **精神/意识形态** | IDEOLOGICAL | 观念层面的凝聚力与合法性 | 名誉、信仰、大义、尊严、历史记忆 |

**数据结构**：

## 三、成住坏空引擎

### 3.1 成（Generation）- 负熵注入

**自然生成**：

$$G_i(x,t) = A_i \times f(\text{weather}, \text{season}) \times \text{productivity}$$

**生成公式示例**：

| 资源 | 生成公式 |
|------|---------|
| 粮食 | $G_{food} = A_{farm} \times R_{rain} \times T_{temp} \times \eta_{agri}$ |
| 能源 | $G_{energy} = A_{mine} \times \eta_{mine} \times \text{tech\_level}$ |
| 模因 | $G_{meme} = \text{creation\_rate} \times \text{population} \times C_{creativity}$ |

**参数定义**：

| 参数 | 含义 | 范围 |
|------|------|------|
| $A_i$ | 产能面积/基数 | $\ge 0$ |
| $\eta_i$ | 效率系数 | $(0, 1]$ |
| $R_{rain}$ | 降水因子 | $[0, 2]$ |
| $T_{temp}$ | 温度因子 | $[0, 2]$ |

### 3.2 住（Stability）- 稳态维持

**稳态判定**：

$$\text{Phase} = \begin{cases}
\text{STABLE} & \text{if } S_i > \theta_{critical} \text{ and } \frac{dS_i}{dt} \approx 0 \\
\text{DECLINING} & \text{if } \frac{dS_i}{dt} < 0 \\
\text{GROWING} & \text{if } \frac{dS_i}{dt} > 0
\end{cases}$$

**稳态维持机制**：

- 资源存量 > 消费阈值时，系统处于"住"态
- 短暂的平衡期，人格引擎有喘息空间
- "住"的持续时间取决于资源缓冲区大小

### 3.3 坏（Decay）- 熵增法则

**这是世界引擎的核心动作！**

**基础衰减**：

$$D_i = \lambda_i \times S_i \times f(\text{environment})$$

**环境加速因子**：

$$f(\text{env}) = (1 + T_{stress}) \times (1 + H_{humidity}) \times (1 - M_{maintenance})$$

**冲突加速**：

$$D_{conflict} = \alpha_{conflict} \times I_{conflict} \times S_i$$

**信息热寂**：

$$D_{info} = \lambda_{info} \times T_{info} \times P_{polarization}$$

**综合衰减公式**：

$$\frac{dS_i}{dt} = G_i - C_i - D_i^{natural} - D_i^{conflict}$$

### 3.4 空（Void）- 相变重置

**相变触发条件**：

$$\text{if } Re_s > Re_{critical} \text{ or } S_{critical} < \theta_{collapse}:$$
$$\quad \text{trigger Phase Transition}$$

**社会雷诺数估算**：

$$Re_s = \frac{\rho \times v \times L}{\mu}$$

其中：
- $\rho$：人口密度
- $v$：冲突速度尺度
- $L$：社会结构特征长度
- $\mu$：社会黏滞度（信任、规范、制度）

**相变类型**：

| 类型 | 触发条件 | 效果 |
|------|---------|------|
| **战乱** | $Re_s > Re_{war}$ | 大规模资源破坏、人口减少 |
| **饥荒** | $S_{food} < \theta_{famine}$ | 人口锐减、社会动荡 |
| **瘟疫** | $I_{disease} > \theta_{plague}$ | 人口减少、贸易中断 |
| **革命** | $P_{polarization} > \theta_{revolution}$ | 政权更迭、制度重置 |

**相变后重置**：

```python
def phase_transition(region, intensity):
    # 打碎旧秩序
    region.earth.food *= (1 - intensity * 0.5)
    region.earth.population *= (1 - intensity * 0.3)
    region.fire.conflict_intensity = min(1.0, intensity * 2)
    
    # 重置部分参数
    region.water.policy_barrier *= 0.5  # 秩序崩塌，壁垒降低
    region.wind.info_temperature = 0.8  # 信息混乱
    
    # 回到"成"的起点
    region.phase = "REBUILDING"
```

## 四、物流系统（水）

### 4.1 运输通量

**单边运输公式**：

$$J_i(u \to v, t) = \min\left( \kappa_i(u,v), H_i \times \frac{S_i(u)}{S_i(u) + S_i(v) + \epsilon} \times (1 - b_{policy}) \right)$$

**参数定义**：

| 参数 | 含义 |
|------|------|
| $\kappa_i(u,v)$ | 容量上限 |
| $H_i$ | 运输努力（车辆/船队/带宽） |
| $b_{policy}$ | 政策壁垒 |
| $\epsilon$ | 数值稳定项 |

### 4.2 运输阻力

**距离阻力**：

$$R_{distance} = \frac{1}{d_{uv}^2 + \epsilon}$$

**政策阻力**：

$$R_{policy} = 1 - b_{policy}$$

**综合阻力**：

$$R_{total} = R_{distance} \times R_{policy} \times R_{trust}$$

### 4.3 掠夺流

**掠夺定义**：非法的资源转移，记为额外损耗项。

$$J_{plunder}(u \to v, t) = \alpha_{plunder} \times I_{conflict} \times S_i(u)$$

## 五、物理-心理映射接口（V10.0 新增）

为了支持人格材质的“合金化”，物理世界必须输出带有**属性标签**的压力（Stress）。

### 5.1 压力张量 (Stress Tensor)

物理事件不再是一个标量强度，而是一个包含“主观应力”与“材质映射”的张量。事件的物理含义由 `stress_type` 和 `impact_vector` 共同决定。

| 压力类型 (Stress Type) | 对应合金元素 (Element) | 物理含义与典型事件 |
| :--- | :--- | :--- |
| **Compressive (压缩)** | **C (碳)** | 持续的生存挤压（如饥荒/贫穷），增加刚度，降低韧性。 |
| **Shear (剪切)** | **S (硫)** | 极端的暴力撕裂（如战乱/屠杀），导致晶格错位（疯狂）。 |
| **Corrosion (腐蚀)** | **P (磷)** | 持续的恐惧侵蚀（如瘟疫/谣言），降低屈服点（意志力）。 |
| **Tensile (拉伸)** | **Ni (镍)** | 需要韧性的长期建设（如灾后重建/被关怀），增加韧性。 |

### 5.2 物理应力计算公式

$$ \sigma_{physics} = \max(|Impact_{SOMATIC}|, |Impact_{MATERIAL}|, |Impact_{SOCIAL}|, |Impact_{IDEOLOGICAL}|) $$

系统提取事件冲击向量中最大的负面影响作为主应力强度，随后交由有情世间的“执念矩阵”计算最终的主观感受。

### 5.3 极端环境广播

当物理区域发生相变（如战乱），系统会向该区域所有实体广播**高浓度合金注入请求**：

```json

  "type": "ENVIRONMENT_INJECTION",
  "region_id": "R01",
  "element": "S",
  "concentration": 0.05,
  "narrative": "战乱爆发，尸横遍野，空气中弥漫着硫磺与血腥味。"
}
```

## 六、信息扩散系统（风）

### 6.1 信息传播模型

**单跳传播强度**：

$$\text{strength} = \text{salience} \times \text{edge\_factor} \times \text{damping\_factor} \times \text{credibility}$$

**边因子**：

$$\text{edge\_factor} = w_{edge} \times \frac{1}{d_{uv}^2 + \epsilon}$$

**阻尼因子**：

$$\text{damping\_factor} = 1 - \lambda_{info}$$

### 5.2 信息扭曲

**极化驱动扭曲**：

$$\text{sentiment}_{next} = \text{clamp}\left( \text{sentiment}_{curr} + \alpha_{distort} \times \text{sharpen}(P) \times \text{bias}, -1, 1 \right)$$

**锐化函数**：

$$\text{sharpen}(P) = \tanh(2 \times (P - 0.5))$$

### 6.3 模因传播

**模因效力**：

$$\text{effect} = \text{potency} \times \text{stock} \times \text{spread\_rate}$$

**模因消费**：

$$\Delta \text{state} = \text{effect} \times f(\text{character\_traits})$$

**局部守恒**：

$$\Delta S_i = G_i - C_i - D_i + \sum_{u} J_i(u \to v) - \sum_{w} J_i(v \to w) + L_i$$

$$\sum_{x} S_i(x, t+1) = \sum_{x} S_i(x, t) + \sum_{x} G_i(x) - \sum_{x} C_i(x) - \sum_{x} D_i(x) + \sum_{boundary} L_i$$
