# 人格模拟系统 10.0 设计方案（下）：群体涌现与社会流体

> **核心哲学澄清**：
> - **微观有情**：单个个体的心理活动（材质、应力、相变）。
> - **宏观有情（共业）**：群体心理产生的场域（信任场、恐慌场、意识形态场）。
> - **器世间**：仅仅是这些场域发挥作用的**物理容器**（资源、地理、人口）。
>
> **误区修正**：群体行为（如挤兑、暴乱、邪教传播）本质上是**“无数个有情众生的心理共振”**，属于**有情世间的高级形态**。器世间只负责记录“死了多少人”、“少了多少粮”，而**“为什么会死人”**、**“为什么会抢粮”**的动力学机制，依然属于有情世间（群体心理学）。

## 一、 社会流体力学 (Social Hydrodynamics)

将群体心理状态视为一种**“流体”**，用流体力学公式模拟社会动荡。

### 1.1 核心参数映射

| 社会心理学概念 | 流体力学对应 | 符号 | 物理/心理含义 |
| :--- | :--- | :--- | :--- |
| **社会信任度 (Trust)** | **黏度 (Viscosity)** | $\mu$ | 阻碍冲突扩散的力量。信任度越高，社会越“粘稠”，冲突越难扩散。 |
| **情绪波动 (Volatility)** | **流速 (Velocity)** | $v$ | 情绪变化的剧烈程度。谣言传播速度越快，流速越高。 |
| **社会密度 (Density)** | **密度 (Density)** | $\rho$ | 人际关系的紧密程度。关系网越密，能量传递越快。 |
| **社会结构 (Structure)** | **管道直径 (Diameter)** | $L$ | 宗族、法律、道德的约束范围。 |

### 1.2 社会雷诺数 (Social Reynolds Number)

判断社会处于**“有序（层流）”**还是**“动荡（湍流）”**的关键指标。

$$ Re_s = \frac{\rho \times v \times L}{\mu} $$

*   **层流状态 ($Re_s < 2000$)**：
    *   **特征**：社会井然有序，冲突被局限在小范围内（法律、道德有效）。
    *   **表现**：范进疯了，只是范家自己的事，邻居最多看个热闹。
*   **湍流状态 ($Re_s > 4000$)**：
    *   **特征**：社会陷入混沌，微小的扰动（谣言）会瞬间引发全服风暴。
    *   **表现**：范进疯了，开始砍人，邻居阿勇被吓到也开始砍人，整个村子瞬间变成修罗场。

### 1.3 动力学方程：纳维-斯托克斯方程的社会学变体

描述“社会动荡”如何在人群中传播：

$$ \frac{\partial v}{\partial t} + (v \cdot \nabla)v = -\frac{1}{\rho}\nabla P + \nu \nabla^2 v + F $$

*   $\nabla P$ (**压力梯度**)：多维资源匮乏的不均程度（贫富差距、权力差距、意识形态冲突）。差距越大，流体运动越剧烈。在张量化沙盒中，$\nabla P$ 是由四大维度（SOMATIC, MATERIAL, SOCIAL, IDEOLOGICAL）的不平衡共同驱动的。
*   $\nu \nabla^2 v$ (**粘性扩散**)：信任和道德对动荡的平抑作用。
*   $F$ (**外力**)：器世间施加的张量化物理打击（饥荒、瘟疫、党争、信仰崩塌）。

## 二、 宏观相变：共业涌现 (Collective Emergence)

当微观个体的“人格合金化”达到一定比例，或者社会雷诺数突破临界值，社会整体发生**宏观相变**。

### 2.1 固态社会 (Solid State) -> 秩序井然
*   **特征**：极高的黏度（$\mu$），极低的流速（$v$）。
*   **结构**：晶格结构稳定（宗族、礼教森严）。
*   **对应历史**：盛世、治世。
*   **人格分布**：绝大多数是“奥氏体”（顺民）。

### 2.2 液态社会 (Liquid State) -> 流动与变革
*   **特征**：黏度降低，流速加快。
*   **结构**：晶格解体，分子自由流动（人口迁徙、思想解放）。
*   **对应历史**：春秋战国、大航海时代。
*   **人格分布**：混合态，出现大量“马氏体”（野心家）。

### 2.3 气态社会 (Gaseous State) -> 暴乱与毁灭
*   **特征**：黏度趋近于 0（信任崩塌），流速极高（情绪极化）。
*   **结构**：无结构，分子高速碰撞（所有人对所有人的战争）。
*   **对应历史**：五胡乱华、文革、末日废土。
*   **人格分布**：全员“马氏体”或“脆性断裂”（疯子）。

## 三、 跨层交互接口：器世间 <-> 有情世间

### 3.1 器世间 -> 有情世间（物理边界条件）
*   **Input**：张量化事件（OpenEvent），包含对四大维度的冲击向量。
*   **作用**：作为流体力学方程中的**边界条件 (Boundary Conditions)** 和 **外力项 ($F$)**。
*   *例子*：器世间减少了 50% 的粮食（SOMATIC 冲击）并且伴随分配不公（SOCIAL 冲击） -> 方程中的压力梯度 $\nabla P$ 瞬间增大 -> 驱动流速 $v$ 暴涨。

### 3.2 有情世间 -> 器世间（共业反噬）
*   **Input**：社会流体状态（层流/湍流）、破坏性动能。
*   **作用**：
    1.  **多维资源损耗**：湍流状态下，四大维度的资源消耗速度暴涨（抢劫粮库消耗 SOMATIC 和 MATERIAL，权威扫地消耗 SOCIAL 和 IDEOLOGICAL）。
    2.  **结构破坏**：当流速 $v$ 超过建筑/制度强度，物理设施或社会制度被摧毁。
    3.  **人口死亡**：高能粒子碰撞（械斗）导致人口直接减少。

## 四、 10.0 完整模拟闭环

1.  **微观层（上篇）**：
    *   饥荒或党争（外力 $F$ 的张量冲击）导致范进（粒子）发生**塑性形变**，变成“马氏体”（高硬度粒子）。
2.  **介观层（统计力学）**：
    *   大量范进变硬，导致社会整体**黏度 $\mu$ 下降**（没人信道德了，IDEOLOGICAL 崩塌）。
3.  **宏观层（下篇）**：
    *   黏度 $\mu$ 下降 + 压力梯度 $\nabla P$ 增大 -> **社会雷诺数 $Re_s$ 突破临界值**。
    *   社会从“固态”瞬间**相变**为“气态”（湍流）。
4.  **反噬层**：
    *   气态社会的高速碰撞摧毁了器世间的粮仓与衙门。
    *   四大维度资源进一步减少 -> 压力梯度 $\nabla P$ 进一步增大 -> **死循环（末日崩塌）**。
