## 摘要

有情世间引擎是虚拟世界的微观心理层，负责模拟"有情众生"的感知、决策与演化。其核心架构基于七步演算管线（触缘受想行识业），将外部事件转化为内部状态变化，并输出决策与业力。本引擎不包含任何世界观或目的论，仅执行心理物理学法则：感知刺激、计算驱动力、权衡内耗、做出决策、演化状态。

## 一、核心理念

### 1.1 有情世间的定义

在佛教宇宙观中，"有情世间"（Sentient World）指具有感知、情感、意志的生命世界。"有情"即有意识、能感受苦乐的众生。

**工程意义**：人格引擎需要模拟完整的心理过程，从感知到决策再到状态演化。

### 1.2 五蕴七步架构

传统佛教的"五蕴"（色受想行识）被扩展为七步演算管线：

| 步骤 | 梵文 | 语义 | 系统功能 |
|------|------|------|---------|
| **Step 1** | Sparśa | 触 | 贝叶斯感知模型 |
| **Step 2** | Pratyaya | 缘 | 情境调节模块 |
| **Step 3** | Vedanā | 受 | 双向感受调节 |
| **Step 4** | Saṃjñā | 想 | 认知-动机系统 |
| **Step 5** | Saṃskāra | 行 | 内耗动力学 |
| **Step 6** | Vijñāna | 识 | 决策动力学 |
| **Step 7** | Karma | 业 | 状态演化系统 |

### 1.3 苦乐驱动与多维执念 (Desire Matrix)

有情众生的核心驱动力是"离苦得乐"：

$$\text{Drive} = f(\text{Suffering}, \text{Hope}, \text{Capability})$$

**苦（Dukkha）的本源**：佛教心理学（《俱舍论》）认为，客观的匮乏（色）本身不会产生苦（受），只有当它与主观的“执念”（贪）结合时，才会产生真正的痛苦应力。
引擎将“执念”具象化为对四大维度的权重向量（Desire Matrix）：
- SOMATIC 执念：对生存和肉体的渴望
- MATERIAL 执念：对财富和物质的贪恋
- SOCIAL 执念：对权力和地位的执着
- IDEOLOGICAL 执念：对信仰、名誉、大义的坚守

## 二、变量字典（升级至 V10.0 材质力学标准）

### 2.1 角色材质属性 (Character Material Properties)

原有的 $C$ 向量被重构为物理材质属性，以支持非线性响应与相变。

| 属性符号 | 材质力学名称 | 心理学/人格含义 | 核心作用 |
| :--- | :--- | :--- | :--- |
| **$E$** | **弹性模量 (Stiffness)** | **心理刚度** | 决定面对压力时的情绪波动幅度。$E$ 越高，越冷血/理性。 |
| **$T$** | **韧性 (Toughness)** | **心理韧性** | 决定能吸收多少创伤而不崩溃。$T$ 越高，越能抗压。 |
| **$\sigma_y$** | **屈服强度 (Yield Point)** | **异化阈值** | 决定何时开始产生不可逆的性格扭曲（塑性形变）。 |
| **$\sigma_f$** | **断裂强度 (Break Point)** | **崩溃阈值** | 决定何时彻底疯癫或自杀。 |

### 2.2 动态状态向量（引入塑性形变）

**应力-应变状态**：
- $\sigma^{(t)}$ (当前心理应力)：由外界事件强度决定。
- $\varepsilon_e^{(t)}$ (弹性应变)：暂时的情绪波动，随压力消失而恢复。
- $\varepsilon_p^{(t)}$ (塑性应变/性格扭曲度)：**新增**，永久累积的性格改变。$\varepsilon_p$ 的增加会导致 $E$ 和 $\sigma_y$ 的漂移。

**合金成分状态**：
- $Elements^{(t)}$ (合金元素表)：记录当前人格中掺杂的异质元素浓度（如 $C_{concentration}$）。

**原有状态保留**：
- $E_{mood}$ (情绪基调) 映射为 $\varepsilon_e$ 的积分。
- $C_{trauma}$ (创伤累积) 映射为 $\varepsilon_p$。

**经验累积**：
- $H^{(t)}(E_{type})$ (对某类事件的经历次数)

### 2.3 事件刺激张量 $E$ (OpenEvent)

不再使用标量表示事件强度，而是将其张量化为四大维度的冲击向量：
- $E_{impact\_vector}$：包含 SOMATIC, MATERIAL, SOCIAL, IDEOLOGICAL 四个维度的浮点数值。正数为增益，负数为剥夺。
- $E_{stress\_type}$：物理应力类型（COMPRESSIVE, SHEAR, CORROSION, TENSILE）。
- $E_{description}$：自然语言描述，由 LLM 世界翻译官负责解析。

### 2.4 关系向量 $B$ (Bond)

- $B_{auth}$ (权威压迫感)
- $B_{intm}$ (亲密度)
- $B_{res}$ (仇恨/怨念值)
- $B_{trust}$ (信任度)
- $S_{chm}$ (发起方的外貌/魅力值)

### 2.5 情境状态向量 $S$ (Situation)

**生理情境**：
- $s_{fatigue}$ (疲劳度)
- $s_{hunger}$ (饥饿度)
- $s_{illness}$ (疾病状态)

**心理情境**：
- $s_{anxiety}$ (焦虑状态)
- $s_{depression}$ (抑郁状态)

**社会情境**：
- $s_{loneliness}$ (孤独感)
- $s_{support}$ (社会支持)
- $s_{belonging}$ (归属感)

## 三、七步演算管线

### Step 1: 触 (Contact) - 贝叶斯感知与主观应力计算

**核心功能**：用贝叶斯概率模型整合先验预期、历史经验，结合执念矩阵计算多维主观应力（$\sigma$）。

#### 1.1 先验概率模型

**综合先验**：

$$P(H_1) = \frac{1}{|T_E|} \times \sum_{t \in T_E} [P_{base}(t) + P_{exp}(t) + P_{personality}(t)] + P_{context}$$

**历史经验修正**：

$$P_{exp}(t) = \alpha \times [1 - \exp(-H(t)/\tau)]$$

#### 1.2 证据生成模型

**信号强度**：

$$\mu_{signal} = (1 - E_{sub}) \times C_{sen} \times (1 - C_{dist} \times \delta_{dist}) \times g(context)$$

**感知噪声**：

$$\sigma^2 = \sigma_0^2 \times (1 + C_{dist} \times \phi_{dist}) \times (1 + s_{fatigue} \times \phi_{fatigue})$$

#### 1.3 贝叶斯更新

**后验概率**：

$$P(H_1 | x) = \frac{P(H_1) \times \exp\left(-\frac{(x-\mu_{signal})^2}{2\sigma^2}\right)}{P(H_1) \times \exp\left(-\frac{(x-\mu_{signal})^2}{2\sigma^2}\right) + P(H_0) \times \exp\left(-\frac{x^2}{2\sigma^2}\right)}$$

#### 1.4 感知决策

**概率性决策**：

$$A \sim \text{Bernoulli}(P(H_1 | x))$$

**置信度**：

$$\text{confidence} = |P(H_1 | x) - 0.5| \times 2$$

### Step 2: 缘 (Conditioning) - 情境缓冲与历史应力集中点放大

**核心功能**：当前情境缓冲应力，历史创伤锚点产生应力集中效应（$K_t$）。

#### 2.1 Simple模式（默认）

**负向事件调节**：

$$\text{neuro\_factor} = 1.0 + C_{neu} \times 0.4$$
$$\Delta_{hurt} = 2.5 \times B_{intm} \times E_{pain} \times \text{intensity}$$

**正向事件调节**：

$$\text{open\_factor} = 1.0 + C_{opn} \times 0.3$$
$$\Delta_{support} = 2.0 \times B_{trust} \times E_{warm} \times \text{intensity}$$

**饱和限制**：

$$M_{final} = 1.0 + \tanh(M_{cond}' - 1.0) \times 3.0$$

### Step 3: 受 (Sensation) - 弹性应变与塑性形变计算

**核心功能**：计算短暂情绪波动的弹性应变（$\varepsilon_e$），以及不可逆的性格扭曲（$\varepsilon_p$）。

#### 3.1 正向感受（强化模型）

**敏感度计算**：

$$S_{sens}^{base} = 1 - \exp\left(-\frac{H^{(t)}(E_{type}^{+})}{\tau_{sens}}\right)$$

**强化效应**：

$$S_{sens} = S_{sens}^{base} \times \left(\frac{\sigma_{applied}}{\sigma_{yield}}\right)^n$$

**正向感受强度**：

$$S_{+} = E_{warm} \times S_{sens}^{final} \times M_{cond}^{final} \times \text{personality\_factor}_{+}$$

#### 3.2 负向感受（损伤模型）

**损伤演化（Kachanov损伤律）**：

$$\dot{D} = A \times \left(\frac{\sigma_{applied}}{1-D}\right)^m$$

**负向感受强度**：

$$S_{-} = E_{pain} \times (1 - D_{eff}) \times M_{cond}^{final} \times \text{personality\_factor}_{-}$$

#### 3.3 净感受

$$S_{net} = S_{+}^{final} - S_{-}^{final}$$

### Step 4: 想 (Cognition) - 动机生成与合金相变判定

**核心功能**：计算认知与动机（马斯洛），并在极端应力下由LLM判断合金注入与相变触发。

#### 4.1 动机层次系统

| 层次 | 需求 | 驱动力公式 |
|------|------|-----------|
| **1** | 生理 | $D_{survival} = S \times (E_{risk} \times 2.0 + E_{pain} \times 1.5)$ |
| **2** | 安全 | $D_{safety} = S \times E_{risk} \times (1 + C_{dou} \times 0.5)$ |
| **3** | 归属 | $D_{belonging} = S \times E_{warm} \times (1 + C_{emp} \times 0.4) \times B_{intm}$ |
| **4** | 尊重 | $D_{esteem} = S \times C_{pri} \times 2.0 \times (1 + E_{evil} \times 0.5)$ |
| **5** | 自我实现 | $D_{growth} = S \times E_{gain} \times C_{opn} \times C_{hope}$ |

#### 4.2 驱动力向量

**负向驱动力**：
- 攻击欲: $D_{agg}$
- 尊严感: $D_{dig}$
- 贪婪欲: $D_{acq}$

**正向驱动力**：

#### 4.3 阻力向量

- 风险阻力: $C_{risk}$
- 道德阻力: $C_{mor}$
- 能力阻力: $C_{cap}$
- 情境阻力: $C_{sit}$

#### 4.4 净驱动力

$$D_{total} = \sum_{i} w_{final}^{(i)} \times D^{(i)} \times \text{context\_factor}$$

$$C_{total} = \sum_{j} w_{final}^{(j)} \times C^{(j)} \times \text{context\_factor}$$

$$F_{net} = D_{total} - C_{total}$$

### Step 5: 行 (Conflict) - 内耗动力学

**核心功能**：计算驱动力与阻力之间的"摩擦力"，O(n)复杂度。

#### 5.1 四维内耗系统

**认知内耗**（思维冲突）：

$$F_{cognitive} = |D_{Top1} - D_{Top2}| \times (D_{total} \times 0.3)$$

**情感内耗**（情绪冲突）：

$$F_{emotional} = \sqrt{\frac{1}{n}\sum_{i}(D^{(i)} - \bar{D})^2} \times C_{neu}$$

**行为内耗**（行动冲突）：

$$F_{behavioral} = \min(D_{total}, C_{total})$$

**价值内耗**（道德冲突）：

$$F_{value} = \max(0, D_{immoral} - C_{mor\_total}) \times C_{csc}$$

#### 5.2 综合内耗

$$F_{base} = F_{cognitive} + F_{emotional} + F_{behavioral} + F_{value}$$

$$\text{personality\_factor} = \frac{1 + C_{neu} \times 0.5}{1 + C_{int} \times C_{wis} \times 0.3} \times \frac{1}{1 + C_{per} \times 0.2}$$

$$F_{int} = F_{base} \times \text{personality\_factor}$$

### Step 6: 识 (Decision) - 五态决策生成

**核心功能**：基于净动力（$F_{net}$）生成五种明确的决策状态枚举，由LLM输出具体行动。

#### 6.1 净动力计算

$$F_{net}^{corrected} = (D_{total} - C_{total}) \times (1 - F_{int} \times 0.3)$$

$$\text{personality\_modifier} = 1.0 + (C_{bra} - 0.5) \times 0.4 - (C_{dou} - 0.5) \times 0.3$$

$$F_{net} = F_{net}^{corrected} \times \text{personality\_modifier}$$

#### 6.2 动态阈值

$$\theta_{personality} = \theta_{base} \times \frac{1.0}{1.0 + C_{bra} \times 0.5} \times (1.0 + C_{dou} \times 0.3)$$

#### 6.3 五态决策系统

| 状态 | 条件 | 心理学意义 |
|------|------|-----------|
| **爆发** | $F_{net} > 2\theta$ | 强烈行动，情绪爆发 |
| **行动** | $\theta < F_{net} \le 2\theta$ | 正常行动，理性决策 |
| **犹豫** | $-\theta \le F_{net} \le \theta$ | 内耗状态，决策困难 |
| **隐忍** | $-2\theta \le F_{net} < -\theta$ | 压抑情绪，理性控制 |
| **崩溃** | $F_{net} < -2\theta$ | 心理崩溃，无法行动 |

#### 6.4 决策置信度

$$\text{confidence} = \frac{|F_{net}|}{|F_{net}| + F_{int}}$$

**核心功能**：输出影响器世间的多维业力，完成时间衰减、状态持久化与PTSD触发。

#### 7.2 时间衰减系统

**精神失真自然衰减**：

$$C_{dist}^{(t+\Delta t)} = C_{dist}^{(t)} \times \exp(-\lambda_{dist} \times \Delta t)$$

**触发概率**：

#### 7.5 多途径恢复系统

**治愈力**：

$$F_{heal} = E_{warm} \times (1 + C_{hope}) \times C_{wis}$$

**演化原则**：
- 稳定性：$\Delta C_{trait} \in [-0.01, 0.01]$

$$C_{neu}^{(t+1)} = C_{neu}^{(t)} + \alpha_{neu} \times (\bar{C}_{trauma} - 0.3) \times \text{plasticity}$$

### 4.2 预期落差转换（相对剥夺感）

将客观冲击向量结合角色的主观预期，转换为心理主观应力：

```python
def calculate_subjective_stress(event, character):
    total_stress = 0.0
    for dim, impact in event.impact_vector.items():
        if impact < 0: # 只有剥夺才会产生痛苦应力
            loss_intensity = abs(impact)
            desire_weight = character.desire_matrix[dim]
            expected_val = character.expected_resources[dim]
            expected_drop = loss_intensity * (expected_val / 100.0)
            
            # 主观应力 = (绝对短缺度 * 渴望度) + (预期落差 * 损失厌恶系数2.25)
            dim_stress = (loss_intensity * desire_weight) + (expected_drop * 2.25)
            total_stress += dim_stress
            
            # 预期适应 (Hedonic Adaptation)
            character.expected_resources[dim] *= 0.9
            
    return total_stress
```
